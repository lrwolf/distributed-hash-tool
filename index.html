• Use the up and down arrow keys to change the number of nodes in the cluster.
    • The keys present in the nodes will be automatically rebalanced across the updated cluser.
  • Pressing any letter (a-z) will do the appropriate operation to the appropriate node based on the mode:
    • Left arrow key : get mode - sends a key to a node to be added (or updated if it is present)
    • Right arrow key : put mode - retrieves a key from a node if it is present
    • Space bar : delete mode - removes a key from a node if it is present
